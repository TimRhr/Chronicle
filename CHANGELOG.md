# Changelog

All notable changes to this project will be documented here.

## [1.0.0] - 2025-12-20

### Highlights
- Complete social blogging experience: posts with Markdown, media galleries, polls, link previews, comments, reactions, bookmarks, notifications, and @mentions.
- Group collaboration: creation, invitations, announcements, shared files, and role management for teams.
- Personalized profiles: theme colors, layout styles (list, grid, masonry, timeline), avatars, cover images, and localized language preferences.
- Admin analytics: dashboards with KPIs, engagement charts, growth tracking, real-time stats, and invite management.
- Secure access: local registration with invites, Keycloak SSO integration, rate-limited auth routes, and password reset emails.
- Production-ready stack: Flask app factory, SQLAlchemy models, Redis caching/limiting, Socket.IO events, Tailwind build pipeline, and Docker Compose deployment.
